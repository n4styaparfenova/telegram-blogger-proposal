# Коммерческое предложение: Совместный Telegram-канал с блогером

## Описание проекта

Профессиональное коммерческое предложение для блогеров с аудиторией 200K-1M+ подписчиков о создании совместного миллионного Telegram-канала.

## Файлы проекта

- `proposal.html` - HTML-лендинг (основной файл)
- `proposal.pdf` - PDF-версия для отправки (см. инструкцию по созданию ниже)
- `README.md` - этот файл с описанием

## Как открыть HTML-версию

Просто откройте файл `proposal.html` в любом современном браузере:
- Двойной клик по файлу
- Или перетащите файл в окно браузера
- Или правой кнопкой → "Открыть с помощью" → выберите браузер

## Как создать PDF-версию

### Вариант 1: Через браузер (самый простой)

1. Откройте файл `proposal.html` в браузере Chrome, Edge или Safari
2. Нажмите `Cmd+P` (Mac) или `Ctrl+P` (Windows)
3. В настройках печати выберите "Сохранить как PDF"
4. Установите параметры:
   - Поля: минимальные
   - Фоновая графика: включить
5. Сохраните как `proposal.pdf`

### Вариант 2: Онлайн-конвертер

Можно использовать любой онлайн-конвертер HTML → PDF:
- https://www.sejda.com/html-to-pdf
- https://cloudconvert.com/html-to-pdf
- https://www.ilovepdf.com/html-to-pdf

### Вариант 3: Через командную строку (для продвинутых)

Если у вас установлен Node.js:

```bash
npm install -g html-pdf-chrome
html-pdf-chrome proposal.html proposal.pdf
```

Или используйте wkhtmltopdf:

```bash
brew install wkhtmltopdf  # для Mac
wkhtmltopdf proposal.html proposal.pdf
```

## Особенности дизайна

- Современный Telegram-friendly дизайн с синими акцентами
- Полностью адаптивный под мобильные устройства
- Визуальные схемы и инфографика
- Таблица с реальными примерами доходности
- Сравнительные блоки "проблема vs решение"
- Яркие акценты на ключевых цифрах
- Время чтения: 3-5 минут

## Основные блоки

1. **Заголовок** - привлекательное описание сути проекта
2. **Кто мы** - экспертиза и опыт
3. **Как это работает** - схема перелива трафика
4. **Почему без лица = больше денег** - сравнение подходов
5. **Кто что делает** - распределение ролей
6. **Реальные цифры с рынка** - таблица доходности
7. **Почему это выгодно вам** - преимущества для блогера
8. **Почему это выгодно нам** - win-win подход
9. **Следующие шаги** - CTA и контакты

## Ключевые цифры

- Инвестиция: 6,3M₽ (по 3,15M₽ каждый)
- Целевая аудитория: 1M подписчиков
- Прогноз дохода: 5-7M₽/месяц
- Окупаемость: 1-2 месяца
